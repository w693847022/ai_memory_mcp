## 阶段 7: 代码实现和测试

**目标**: 按照选定方案实现功能

**建议使用auto模式**

**流程**:
1. 按方案步骤实现代码
2. 每完成一个步骤，更新进度显示
3. 实现完成后运行测试用例并确保测试用例成功

**🔍 自动检测**:
```
检测内容: 代码已按方案实现，测试全部通过
检测方式:
  1. 代码文件存在性检查: 检查方案中涉及的文件是否已创建/修改
  2. 测试执行: 运行相关测试用例
检测通过标准:
  ✓ 方案中涉及的核心文件已创建或修改
  ✓ 相关测试用例执行通过
  ✓ 无语法错误或导入错误
```

**检测失败处理**: ⏸️ **检测失败，暂停等待用户处理**
- 失败原因: 代码文件未创建/修改，或测试用例执行失败
- 处理方式: 请用户修复代码问题或测试问题，然后确认继续
